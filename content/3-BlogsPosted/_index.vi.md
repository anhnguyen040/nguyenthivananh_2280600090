Tại đây sẽ là phần liệt kê và giới thiệu các blogs mà bạn đã dịch.

###  [Blog 1 - Automating CI/CD for GitHub Monorepos with AWS CodePipeline](3.1-Blog1/)
Bài viết này trình bày cách xây dựng CI/CD cho GitHub monorepo bằng AWS CodePipeline, API Gateway và Lambda. Bạn sẽ hiểu cách thiết kế một lớp quyết định đường dẫn thay đổi, chỉ khởi động các pipeline liên quan khi cần thiết, từ đó giảm chi phí build, rút ngắn thời gian phản hồi và cải thiện trải nghiệm phát triển.

###  [Blog 2 - Triển khai Modern Data Platform trong vài phút với MDAA](3.2-Blog2/)
Bài viết này giới thiệu AWS Modern Data Architecture Accelerator (MDAA) và cách nó giúp đơn giản hóa việc triển khai Modern Data Platform. Nội dung đề cập kiến trúc module, governance được tích hợp ngay từ đầu, và cách AWS chuẩn hoá triển khai data lake, analytics và nền tảng AI để vận hành an toàn, khả dụng và dễ mở rộng.

###  [Blog 3 - Xác thực hai lớp cho Nakama Game Server với Amazon Cognito](3.3-Blog3/)
Bài viết này mô tả kiến trúc xác thực hai lớp giữa Amazon Cognito và Nakama. Nó trình bày luồng JWT Cognito, cách Nakama tạo Session Token qua Go Runtime Hook, và vai trò của CloudFront, ALB, NLB, ECS Fargate trong bảo mật và hiệu năng hệ thống.
