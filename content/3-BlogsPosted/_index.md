###  [Blog 1 - Automating CI/CD for GitHub Monorepos with AWS CodePipeline](3.1-Blog1/)
This post explains how to build a path-filtered CI/CD workflow for a GitHub monorepo using AWS CodePipeline, API Gateway, Lambda, and a decision engine. It shows how to run only the relevant pipeline when a commit changes a specific service folder, reducing build cost, eliminating unnecessary builds, and providing faster feedback for developers.

###  [Blog 2 - Deploy Modern Data Platforms in Minutes with MDAA](3.2-Blog2/)
This post introduces AWS Modern Data Architecture Accelerator (MDAA) and how it accelerates Modern Data Platform deployment. It highlights module-based infrastructure, governance by design, and how AWS best practices simplify data lake, analytics, and AI platform builds on AWS.
